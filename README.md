## Getting Started

### Prerequisites


* Docker


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iotaledger/evm-toolkit.git
   ```
3. Build image
   ```sh
   docker build -t evm-toolkit .
   ```
4. Run toolkit
   ```sh
   docker run -p 127.0.0.1:8080:80 evm-toolkit
   ```
